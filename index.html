¡Bienvenida a Alerta Ella!

Este es un espacio seguro creado y pensado únicamente para mujeres

Alerta Ella es una app móvil gratuita creada con Python, dedicada exclusivamente a mujeres que se sienten en peligro. Con un solo botón, pueden enviar una alerta silenciosa a sus contactos de confianza o autoridades locales, incluyendo su ubicación en tiempo real.

Este proyecto nace como homenaje, protección y acto de justicia emocional. Cada pantalla, cada ícono, cada línea de código está pensada para transmitir seguridad, empatía y urgencia silenciosa.
